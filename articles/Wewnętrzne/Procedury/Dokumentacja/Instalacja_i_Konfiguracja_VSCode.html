    Visual Studio Code Instalacja i konfiguracja

    Opis narzędzia

    Visual Studio Code jest zaawansowanym i przyjaznym użytkownikowi edytorem plików tekstowych, który dzięki bogatemu zestawowi rozszerzeń potrafi zaspokoić potrzeby praktycznie każdego użytkownika.

    Instalacja VS Code

    1. W przeglądarce przechodzimy do strony i wybieramy User Instaler x64 Dla Windows

    2. Uruchamiamy instaltor i zostawiamy język Angielski

    3. Akceptujemy licencję

    4. Wybieramy folder instalacji

    5. Wybieramy nazwę w Menu Start

    6. Wybieramy dodatkowe parametry

    7. Przeprowadzamy instalcje

    8. Gotowe - uruchamiamy VS Code w celu dalszej konfiguracji

    Instalacja dodatku sprawdzania języka polskiego

    1. Aby przejść do zarządzania rozszerzeniami używamy sktóru ctrl+shift+x albo klikamy na lewym pasu ikonę

    2. Wpiszujamy wyraz polish i w znalezionym rozszerzeniu Polish Support for LenguageTool klikamy VSCInstallStep1

    3. Musimy jeszcze zainstalować (może już być zainstalowany) dodatek LanguageTool for Visual Studio Code więc wpisujemy jego nazwę do wyszukiwania

    4. Przechodzimy do ustawień za pomocą skrótu ctrl+, bądź korzystająć z ikony zarządzania na lewym pasu i wybierając opcje Settings

    5. W wyszukiwaniu wpisujemy languageTool Ustawiamy język na pl - gotowe.

    6. Jeżeli na komputrzenie nie ma zainstalowanej Java (min wersja 8.0) należy doinstalować -> Instaltor

    Instalacja Git for Windows

    1. Instalujemy na komputerze Git for Windows

    2. Akceptujemy licencje

    3. Wybieramy katalog instalacji

    4. Zostawiamy domyślnie zaznaczone opcje

    5. Wybieramy folder w menu start

    6. Wybieramy VS Code jako domyślny edytor dla Git

    7. Zostawiamy domyślną opcje używania git w lini poleceń

    8. Zostawiamy domyślną opcje dla protokołu HTTPS

    9. Zostawiamy domyślną opcje konwersji końca lini

    10. Ustawiamy linie poleceń widnows

    11. Dodatkowe opcje zostawiamy domyślne

    12. Nie włączamy eksperymentalnych opcji i instalujemy i gotowe.

    Podłączenie do repozytorium dokumentacji git

    1. Przechodzimy w przeglądarce do witryny

    2. Używamy przycisku Colne w prawej sekcji witryny i klikamy przycisk Clone in VS Code

    3. Potwierdzamy otwarcie VS Code

    4. Zezwalamy na otwarcie url

    5. Wyniremalu folder trzymania repozytorium np. D:\repos\dokumentacja i klikamy przycisk select reposytory lokalization

    6. Nastąpi krótkotrwały proces klonowania repozytorium po jego zakończeniu wybieramy opcję Open

    7. Gotowe - możemy pracować na repozytorium

    8. Uruchamiamy VS Code i z Menu Terminal uruchamiamy NewTerminal można zastosować sktót ctrl+shift+`
